# Box Office Mojo Top 200 Movies Scraper

This project uses Selenium to scrape the top 200 movies from Box Office Mojo based on lifetime worldwide gross.

## Features
- Scrapes movie rank
- Scrapes movie title
- Scrapes lifetime gross
- Scrapes release year
- Saves data into CSV format

## Technologies Used
- Python
- Selenium
- Pandas
- WebDriver Manager

## Output
The script generates:

`top_200_movies.csv`
